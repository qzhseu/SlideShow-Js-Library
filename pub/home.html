Getting Started

The most common slide.js use case is to have a single presentation which covers the full viewport. We also support running multiple presentations in parallel on the same page.

Single Presentation

  1. Download slide.js from the github link.

  2. Link to jQuery and the library along with the styles.

     <link rel="stylesheet" href="/css/slideDance.css">
     <link rel="stylesheet" href="/css/theme/black.css" id="theme">
     <script defer src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
     <script defer type="text/javascript" src="./js/slideDance.js"></script>

  3. Create a "SlideDanceGenerator" and call the create the function.

    const sg=new SlideDanceGenerator();
    sg1.create()

  4. Now create your slide using slide.js. Wrap your slides with a div, the class attribute of the div is slideDance. Each slide should inside <slide> tag. 

    <div class="slideDance">
    <div class="slides">
        <slide>
            ...
        </slide>
    </div>
</div>

Multiple Presentation

    To run multiple presentations side-by-side on the same page you can create instances of the SlideDance class. The SlideDanceGenerator accepts an optional argument, the .slideDance HTML element root of the presentation. 

    <div class="slideDance slide1">...</div>
<div class="slideDance slide2">...</div>

<script>
    let slide1=document.querySelector('.slide1');
    const sg1=new SlideDanceGenerator(slide1)
    sg1.create()

    let slide2=document.querySelector('.slide2');
    const sg2=new SlideDanceGenerator(slide2)
    sg2.create()
</script>

If you want to show code block with code syntax highlight in your slide, you can use external modules like highlight.js. Otherwise your code will be displayed in plain text.

API

Config Options

You can create the slides by calling create function. The create function acccepts an optional argument, a slide.js config object. 

sg1.create({
    progress : true,
    controls : true,
    ...
})

The accepted options are the following.

Property Default Type Description
controls true Boolean Whether to allow users to use control buttons to control the slideshow.
controlsPosition bottom-right String This supports two options: 'bottom-right' and 'edges'.
bottom-right: Place the control buttons at the bottom right corner.
edges : Place the control buttons on the four sides of the slides.
controlsTutorial true Boolean Whether to show control tutorial(control buttons will bounce).
progress true Boolean Whether to show progress bar
keyboard true Boolean Whether to allow users to use keyboard to control the slideshow.
editMode false Boolean Whether to allow users to edit slides during presentation. The supported editing operations include drag, resize, rotate, zoom in and zoom out.
showNumberOfPage true Boolean Whether to show number of page.
autoplay true Boolean Whether to autoplay slides.
autoplayTimeInterval 3000 Int The time interval for automatic playback. (in ms).
fontFamily 'Times New Roman', Times, serif String The font style for the whole slides

CallBacks

You can set your callback functions

Property Default Type Description
onReady false Function Triggered when the presentation is initialized.
onSlideChanged false Function Triggered when the slide changes each time .

Methods

Methods to control navigation and check the current state of a presentation.

Navigation

//Start the presentation
slideDanceInstance.startPresentation();

// Navigate to a specific slide
slideDanceInstance.goToSlide(indexh, indexv, indexf)

// Navigate to a horizontal slide
slideDanceInstance.goToHorizontalSlide(indexh);

// Navigate to a vertical slide at current vertical direction.
slideDanceInstance.goToVerticalSlide(indexv);

// Relative slide navigation
slideDanceInstance.nextHorizontalSlide();
slideDanceInstance.preHorizontalSlide();
slideDanceInstance.preVerticalSlide();
slideDanceInstance.nextVerticalSlide();

// Navigate to the first slide
slideDanceInstance.firstSlide();

//Navigate to the last slide
slideDanceInstance.lastSlide();

// Navigate to a fragment on the current slide
slideDanceInstance.goToVerticalSlide(indexv);

//Relative fragment navigation
slideDanceInstance.nextFragment();
slideDanceInstance.preFragment();
slideDanceInstance.upFragment();
slideDanceInstance.downFragment();

// Checks which directions we can navigate towards
// This returns a object like "{top: false, right: true, bottom: false, left: false}"
slideDanceInstance.availableRoutes();

Slides

// Reset all slides to original status
slideDanceInstance.resetSlide();

// -Get the number of total slides"
slideDanceInstance._totalSlides;

// -Get the number of viewed slides.
slideDanceInstance._pastSlideCount;

// -Get current slide
slideDanceInstance._currentSlide;

// -Get the current horizontal slide
slideDanceInstance._currentHorizontalSlide;

// -Get the current vertical slide
slideDanceInstance._currentVerticalSlide;

// -Get all fragments at the current slide
slideDanceInstance._currentSlideFragmentList;

Modes

//Toggle presentation states, optionally pass "true/false" to force "on/off"
//Toggle autoplay mode
slideDanceInstance.toggleAutoSlide();

//Pause autoplay
slideDanceInstance.autoPlayPause();

//Toggle edit slide mode
slideDanceInstance.toggleEditMode();

//-Get the mode state
slideDanceInstance._isAutoSliding;
slideDanceInstance._isPaused;
slideDanceInstance._isEditedMode;

Misc

//Update the status of the control buttons after changing the slide.
slideDanceInstance.updateControlButtons();

//Update the status of the progress bar after changing the slide.
slideDanceInstance.updateProgressBar();

//Get the value of progress bar
slideDanceInstance.getProgressPercentage();

//Update the status of the current page number after changing the slide.
slideDanceInstance.updateNumberOfPage();

//Selected a element on presentation to drag, resize and rotate. The selected element will be wrapped with a colorful border.
let e=document.querySelector('...');
slideDanceInstance.selectElement(e, true);

Complete configuration

Complete configuration with callbacks set.

const sg=new SlideDanceGenerator()
sg.create({
    controls : true,
    controlsPosition : 'bottom-right', //'edges' or 'bottom-right',
    controlsTutorial : true,
    progress : true,
    keyboard : true,  
    editMode : true,
    showNumberOfPage : true,
    autoplay : false,
    autoplayTimeInterval: 2000,
    fontFamily : " 'Satisfy', Helvetica, sans-serif ",

    onSlideChanged: function(){
        console.log('SlideDance: Slide change');
    },

    onEnd: function(){
        console.log('SlideDance: End');
    }
})
