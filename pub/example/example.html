Horizontal Slides and Vertical Slides

Your slides are jumped between using a horizontal sliding transition by default. These horizontal slides are considered as top-level slides in your deck.
You can also nest multiple slides within a single horizontal slide to create a vertical stack. This is convenient to group similar content together in your presentation. When presenting, you can the left/right arrows button to jump through the horizontal slides. When you arrive at a vertical stack you can use up/down arrows to jump through the vertical slides. If you keep clicking the left/right arrow, you will skip the vertical stack.

Here is a simple vertical slide stack

<slide>
    <h4>Slide 1</h4>
</slide>
<slide>
    <slide>
        <h4>Slide 2.1</h4>
    </slide>
    <slide>>
        <h4>Slide 2.2</h4>
    </slide>
</slide>

Fragments

Fragments are used to incrementally present content on your slide. Each fragment with a fragment tag name will appear one by one when you click the right/left arrows. 

<slide>
    <fragment> content1 </fragment>
    <fragment> content2 </fragment>
    <fragment> content3 </fragment>
</slide>

Edit Mode

The presentation can be configured to allow users to enter editing mode while presenting. Try to press "SHIFT" and use mouse to click the element in the slide at the same time. Then you can enter editing mode.

After the selected box appears, you can drag it to any position, adjust the size by dragging the circles at the four corners, and click the circle in the top middle to rotate it. Press "SHIFT" and click other element to switch element. You can press o to exit the edit mode. 

slideDanceInstance.create({ editMode : true })

Auto-Slide

The presentation can be configured to go through slides automatically. To enable this you will need to assign true to "autoplay" and set "autoplayTimeInterval" in the config option. The interval is provided in milliseconds. 

slideDanceInstance.create({ 
    autoplay : true,
    autoplayTimeInterval: 2000,
})

Presenting Code

You can use slide.js to present code and highlight specific lines of code. But you need to replace < and > with < ; and > ; in your code.

If you want to show code block, your code will be displayed as plain text on the slide. But if you want to present code with code syntax highlight, you can use external modules like highlight.js. and so on.

To show your code block you need warp your code with <pre> and <code> You also need to add 'data-trim' attribute to <code>, When the data-trim attribute is added, all surrounding whitespace within the <code> will be removed.

If you want to highlight some lines of code. You can add 'data-hightlight-line' attribute to <code> and provide a comma seperated list of numbers. For example, data-hightlight-line="3,13-17" means line 3, and 13 to 17 is highlighted. 

<pre><code data-code-trim data-hightlight-line="3,13-17">
    import React, { useState } from 'react';

    function ExampleOne() {
        const [time, setTime] = useState(0);
        return (
            <h2> Time: {time} </h2>
        );
    }

Set Background

You can add full-page background on the slide by adding a data-background attribute to your <slide> elements. Slide.js supports three types of backgrounds: color, image and vedio.

Color Background

All CSS color formats are supported, including hex values, keywords or rgba().

<slide data-background="#163a22">
    <p>🦁</p>
</slide>
<slide data-background="lightskyblue">
    <p>⛵</p>
</slide>

🦁

⛵

Image Background

Available attribute options:

Attribute Default Type Description
data-background String Image URL or GIF URL
data-background-repeat no-repeat String Sets how background images are repeated.
data-background-size cover String Sets the size of the element's background image.
The background image will be resized to cover the entire slide by default.

You can wrap your content with a background-content tag and your content will be displayed on a translucent panel. 

<slide data-background="https://images....">
    <background-content>
        <p>You can add content under <italic>background-content tag</italic></p>
    </background-content>
</slide>
<slide data-background="https://images...." data-background-repeat="repeat" data-background-size="300px" >
    <h4>TILED BACKGROUNDS</h4>
</slide>

Video Background

Video will cover the full slide

Setting data-background-video attribute with the video URL.

<slide data-background-video="Your video path">
    <background-content>
        <h4>VIDEO BACKGROUND</h4>
    </background-content>
</slide>
